Parsing

Usage

Let's assume we have a file whose contents look like so:

file=example.txt
author=me
package=zenderer
// here is the point:
values1=stuff,morestuff,finalstuff
values2=data:moredata:finaldata

And we want to retrieve the values after values1 and values2 separately, since zen::util::CINIParser::GetValue() with "values1" as an argument would return "stuff,morestuff,finalstuff". We can thus use zen::util::CINIParser::GetValues() to achieve the effect we want, like so:

zen::util::CINIParser Parser;
Parser.LoadFromFile("example.txt");
// Delimited defaults to a comma (',') character
std::vector<string_t> keyValues = Parser.GetValues("values1");
for(auto& i : keyValues)
{
std::cout << i << ' ';
}
keyValues = Parser.GetValues("values2", ':');
for(auto& i : keyValues)
{
std::cout << i << ' ';
}

The result would be:

stuff morestuff finalstuff data moredata finaldata
