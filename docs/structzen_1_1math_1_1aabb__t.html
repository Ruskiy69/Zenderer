bool zen::math::aabb_t::collides ( const vector_t  tri[3])
inline

This uses the separation of axis theorem (SAT) to perform collision detection. It tests the 3 axes of the triangle in turn, exiting the function as soon as a collision is impossible.

Worst case, the algorithm will perform: 19 multiplications 33 addition / subtractions 25 comparisons

Best case, the algorithm will perform: 7 multiplications 27 addition / subtractions 4 comparisons
