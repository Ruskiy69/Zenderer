bool zen::obj::zAnimation::AddPrimitive ( const gfx::zPolygon &  Prim)
inlinevirtual

This will store a copy of the primitive locally, to prevent bad references occuring later. Thus it's advisable to delete heap memory when done here, or ensure that the given primitive will go off the stack soon, to prevent unnecessary memory duplication.

Parameters
PrimA primitive to make "high-level".
Returns
true if the primitive loaded with a valid textre, and false otherwise.
Note
Currently, multiple calls to this method will stack primitive on top of one another, since there is no position specification on them.
