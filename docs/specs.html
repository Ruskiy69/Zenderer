File Specifications

Formatting specifications for various file types used in Zenderer.

ZenLevel

Specification

Light Options

All floating point values can have a maximum of three decimal places of precision in them. fp – floating-point

Ambient Lights:

  • type – 0
  • color – 3 fp values
  • brightness – 1 fp value

Point Lights:

  • type – 1
  • color – 3 fp values
  • brightness – 1 fp value
  • attenuation – 3 fp values

Spot Lights:

  • type – 1
  • color – 3 fp values
  • brightness – 1 fp value
  • attenuation – 3 fp values
  • maxangle – 1 fp value
  • minangle – 1 fp value

Example File

 filename=sample.zlv
 title=Level 1 -- Tutorial
 // Mesh from existing file
 <entity>
   position=100,200
   filename=grasstile.znt
 </entity>
 // In-line mesh creation of a grass sprite tiled 4 times.
 <entity>
   position=100,100
   vertices=0,0,64,0,0,64,64,64
   texcoords=0,2,2,2,2,0,0,0
   <surface>
     texture=grass.tga
   </surface>
 </entity>
 <light>
   // 2 --> spotlight
   type=2
   color=0.050,0.762,0.556
   attenuation=0.001,0.005,0.000
   position=100,200
   minangle=45.0
   maxangle=225.0
 </light>
