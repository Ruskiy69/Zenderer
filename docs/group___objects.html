Detailed Description

These classes are a step above the simple primitive rendering scheme set up by the architecture in zen::gfx::zPolygon, and are the most likely to face direct user interaction through the application.

Data Structure Documentation

struct zen::obj::bbox_t

This is used for basic quad collision detection throughout Zenderer using the separation of axis theorem. The two components are defined by the top-left and the bottom-right points of a standard, unrotated quad.
